# Simple Python Calculator

## Description
This project is a basic calculator implemented in Python. It is designed to perform four fundamental arithmetic operations: addition, subtraction, multiplication, and division. The program runs entirely in the terminal and provides a straightforward menu‑driven interface for users. Its main goal is to demonstrate simple input handling, function definitions, and loop control in Python.

## How it works
1. The program displays a menu with five options (1. Addition, 2. Subtraction, 3. Multiplication, 4. Division, 5. Exit).
2. The user selects an operation by entering the corresponding number.
3. The program prompts the user to enter two numbers.
4. If the input is valid, the program calls the appropriate arithmetic function, which performs the calculation and prints the result. Division includes a check to avoid division by zero.
5. The loop continues until the user chooses option 5, at which point the program exits.

## Status
> This project is currently functional and is being maintained for educational purposes. Future improvements may include better input validation and support for more operations.
